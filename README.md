# Wordle_Devesh_Klis
# Projet Wordle en C

# Description

Ce projet est une version simplifiée du jeu Wordle développée en langage C.
Le joueur doit deviner un mot de 5 lettres en 6 essais maximum.

# Installation

```bash
git clone <url_repo>
cd wordle_nom1_nom2
```

# Compilation

```bash
gcc src/main.c -o wordle
```

# Exécution

```bash
./wordle
```

# Règles

* 6 essais maximum
* Mot de 5 lettres
* Indications :

  * Lettre correcte et bien placée
  * Lettre présente mais mal placée
  * Lettre absente

# Auteurs

* Devesh
* Klis
